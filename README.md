# 2310 Docker Example

This project highlights how Docker can be useful in the deployment of an application.

## Container

The container can be built with:

```shell 
docker build -t notes .
```

or

```shell 
docker-compose build
```

---

To bring up the stack, run:

```shell 
docker-compose up -d
```

---

And stop it with:

```shell
docker-compose stop
```
